# Coffee Shop Simulator

A Python command-line application that simulates the operations of a digital coffee vending machine. The program manages inventory resources, processes coin payments, calculates change, and serves various coffee beverages based on real-time ingredient availability.

## Features

- **Interactive Menu**: Choose from Espresso, Latte, Capuccinno, Cold Coffee, and Hot Coffee.
- **Resource Management**: Tracks levels of water, milk, and coffee beans, automatically blocking orders if ingredients are insufficient.
- **Coin Processing**: Accepts quarters, dimes, nickels, and pennies to calculate total payments.
- **Transaction Validation**: Verifies if the inserted amount covers the beverage cost and returns precise change.
- **Admin Commands**: Secret triggers to generate resource reports (`report`) or shut down the machine (`off`).
